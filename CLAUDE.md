## Project Overview

UTCLA (United Tribal Countries Land Alliance) is a voluntary alliance of self-governing Tribal Nations across Terra Australis. This repository houses the Alliance's digital presence and documentation.

## Repository State

This is a newly initialized repository. The foundational document is `UTCLA_INFO.md`, which contains the Alliance's full governance framework, vision, and structure.

## Key Context

- **Tjukurpa** (Law/Lore) is the supreme cultural authority and governance foundation — all work in this repository should respect and reflect this principle.
- The Alliance coordinates between autonomous, self-governing Member Countries/Nations — it does not replace their sovereignty.
- Founding members: Pintupi Wati Tjuta Tjungarinyi, Ni'ripono Gwata Tu Sovereign Government, and Majagi.
- Governance uses equal male/female representation per Country/Nation with consensus-based decision-making.
